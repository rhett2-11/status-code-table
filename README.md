# status-code-table
I began this project by setting up all of the tools I needed, such as: downloading VS Code, logging into GitHub to create a new repo, and installing the Live Server extension.

Once I had everything ready to begin development, I started thinking through the ways of parsing the svg data table. I spent time using different svg tools on the web, asked ChatGPT many questions about the file and its source code, and scoured the internet. After some time spent feeling like I was spinning my wheels on the file itself, I scanned the QR code on the svg image and played around with the Troubleshooting Guide fo High Tier Variable Gas Furnace, which appears to have been created by Carrier along with the PDF flowchart.

The existing webapp, along with the PDF flowchart, helped me finally understand the svg data table and ways to troubleshoot these gas furnaces. I started thinking through the code I would need to recreate a simple version of this and began feeling overwhelmed.

It's been years since I wrote my last lines of code, and the anxiety started coming back to me.

I decided to take a break and think through whether web development was really something I wanted to pursue again.

I concluded that I would like to leave my web dev career behind me and pursue more "hands on" work at this point in my life.
